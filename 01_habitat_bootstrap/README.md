# Lesson 01: Habitat Bootstrap

The colony does not have a runtime yet.

There is only one habitat, one resource ledger, and one immediate question:

> can we model the operational truth clearly before we turn it into a process?

This first lesson keeps oxygen, water, power, crew count, and maintenance
updates explicit through pure functions.

## What You'll Learn

- how to model domain state with a struct
- how to keep resource transitions explicit and testable
- why pure state changes still matter before OTP enters the picture
